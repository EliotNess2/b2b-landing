# B2B Landing Page – SEA / Demande de devis

## Objectif
Créer une landing page dédiée Google Ads pour transformer un clic en demande de devis.

## Contexte
Exercice SEA pour un service de développement web.
La landing n’est pas une homepage : elle est conçue autour d’un seul objectif de conversion.

## Mot-clé principal
devis développement web

## Intention
Transactionnelle

## Principes de conversion
- cohérence annonce → landing
- un seul objectif
- CTA visible
- contenu centré sur la demande de devis
- performance et mobile-first
- page de remerciement pour le suivi de conversion

## Stack
- Next.js
- TypeScript
- Tailwind CSS

## Structure de la landing
- Hero avec H1 aligné au mot-clé
- Bloc de preuve / technos
- Formulaire de demande de devis
- FAQ de réassurance
- Page `/merci`

## Ce que ce projet démontre
- compréhension du SEA et de l’intention utilisateur
- capacité à concevoir une landing orientée conversion
- structuration d’un projet front moderne
- logique business + technique

## Lancer le projet
```bash
npm install
npm run dev
